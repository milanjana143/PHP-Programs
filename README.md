## 🐘 Introduction to PHP

**PHP (Hypertext Preprocessor) is a server-side scripting language** used for developing dynamic and interactive websites. It is widely used for web development and works well with databases like MySQL.

PHP was created by **Rasmus Lerdorf** in 1994. It is easy to learn and can be embedded directly into HTML to build dynamic web pages.

### Features of PHP:

* Open-source and free  
* Easy to learn and use  
* Server-side scripting language  
* Supports database integration (MySQL, etc.)  
* Cross-platform compatibility  

### Uses of PHP:

* Dynamic website development  
* Server-side scripting  
* Database-driven applications  
* Web application development   <br><br>
